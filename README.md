# kollekt-erlang

Yet another implementation of the kollekt service (origin by [Paul Asmuth](https://github.com/paulasmuth)).

## Usage

### Prerequisites

Install Erlang (I build with R15B).

### Compile

```shell
./recompile.sh
```

### Run

Demo service:

```shell
./run.sh
```

Test emitter:

```shell
./run_emitter.sh
```
